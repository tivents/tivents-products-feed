Display your products from tivents inside your site with a shortcode via api

== Description ==

Display your products from tivents inside your site with a shortcode. You can decide if you want to present only coupons.
You need a partner account (https://tivents.de/veranstalter) in order get all your products from the tivents api

== Installation ==

1. Upload `tivents-products-feed` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What do i need in order to use this plugin? =

You need a partner account on tivents and your Partner ID. You con get more informations on https://tivents.info/faq

= Can I customize something =

Yes can decide if you want to show all products, coupons or events and you can define the primary and secondary color.
