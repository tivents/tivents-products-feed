Display your products from tivents inside your site with a shortcode

== Description ==

Display your products from tivents inside your site with a shortcode. You can decide if you want to present only coupons


== Installation ==

1. Upload `tivents-products-feed` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
